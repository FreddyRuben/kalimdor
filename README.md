# Kalimdor

English

API to upload files, focused on movie upload.

## Requirements
~~~
node 10 or later
mongoDB 4.0 or later
~~~

## Install
~~~
npm install
~~~

## Run
~~~
npm run dev
~~~

#### Recommendation
Use Postman or Insomnia to test APIs
